## 🎨 CSS Project IA

## 📌 Sobre o Projeto
O CSS Project IA é um projeto front-end desenvolvido com foco em estilização e estrutura visual, utilizando HTML e CSS como principais tecnologias. A proposta foi praticar construção de layout moderno e aplicação de conceitos visuais, explorando organização de componentes e boas práticas de estilização.

O projeto reforça a importância do CSS na criação de interfaces atrativas, organizadas e responsivas.

## 🖥️ Ambiente
Aplicação 100% front-end, executada diretamente no navegador. Pode ser aberta localmente ou hospedada em plataformas como GitHub Pages. Não depende de back-end ou bibliotecas externas.

## 🎯 Objetivos

Praticar criação de layouts modernos com CSS

Trabalhar organização de estrutura HTML

Aplicar conceitos como alinhamento, espaçamento e hierarquia visual

Evoluir a qualidade visual dos projetos de portfólio

## ⚙️ Tecnologias Utilizadas

HTML — Estrutura da página

CSS — Estilização, layout e organização visual

## 🧠 Lógica utilizada
Embora não envolva lógica de programação complexa, o projeto aplica lógica estrutural e visual na organização dos elementos, uso de classes, posicionamento, responsividade e composição do layout.

## 🎓 Aprendizados

Estruturação semântica do HTML

Organização de estilos CSS

Criação de layouts mais profissionais

Evolução na construção visual de interfaces

## ⚠️ Aviso
Este projeto é focado em prática de estilização e estrutura visual. Contém funcionalidades dinâmicas com JavaScript porém é necessário usar sua própria API Key.
